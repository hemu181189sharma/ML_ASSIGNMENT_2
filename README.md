# Breast Cancer Classification ML Project

## Problem Statement
This project predicts whether a tumor is malignant or benign using multiple machine learning classification models.

## Dataset Description
- Dataset: Breast Cancer Wisconsin (UCI)
- Instances: 569
- Features: 30 numeric features
- Target:
    - 0 = Malignant
    - 1 = Benign

## Models Implemented
1. Logistic Regression
2. Decision Tree
3. KNN
4. Naive Bayes
5. Random Forest
6. XGBoost

## Deployment
Deploy using Streamlit Community Cloud.
